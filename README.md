# Blog Platform with Comments

A full-stack Blog Platform with Comments developed as part of an internship project.

This application allows users to create, manage, and interact with blog posts through comments. The platform provides secure user authentication and complete CRUD operations for blog management.

# Features

## User Authentication

- User registration
- User login
- Password encryption
- JWT-based authentication
- Secure user sessions


## Blog Management

Users can:

- Create new blog posts
- View all blog posts
- Edit existing posts
- Delete posts


## Comment System

Users can:

- Add comments to posts
- View comments on blog posts
- Interact with other users' content


## Database Integration

- Stores user information
- Stores blog posts
- Stores comments
- Maintains user-post relationships
